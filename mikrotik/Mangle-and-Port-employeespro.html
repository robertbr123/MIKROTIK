Mangle and Port Net Monitoring for employees pro - MikroTik Script RouterOS

The application allows you to see what is currently displayed on an employee computer monitor. This way you can quickly discover if your employees are doing the tasks they are paid for. All employee computer screens can be displayed at once since they can be displayed as thumbnails with configurable size.

Port forward Net Monitoring for employees pro:
TCP/UDP: 4495/4495


/ip firewall mangle
add action=mark-connection chain=prerouting dst-port="4495" new-connection-mark=conn-employeespro protocol=tcp
add action=mark-connection chain=prerouting dst-port="4495" new-connection-mark=conn-employeespro protocol=udp
add action=mark-packet chain=prerouting connection-mark=conn-employeespro new-packet-mark=employeespro_pkt-up passthrough=no src-address=192.168.1.0/24
add action=mark-packet chain=prerouting connection-mark=conn-employeespro dst-address=192.168.1.0/24 new-packet-mark=employeespro_pkt-down passthrough=no
